About

The Breath Counting App (BCA) is a mobile application that uses real-time audio analysis to detect your breathing patterns and guide you through structured breathing exercises inspired by the Wim Hof Method.

How It Works

4-Phase Breathing Cycle:
  1. Breathing Phase - Count 20-50 breaths with periodic audio signals
  2. First Breath Hold - User-controlled duration, auto-detected
  3. Recovery Breath Hold - Timed hold (10-30 seconds) after deep inhale
  4. Exhale & Cycle Completion - Deep exhale ends cycle

Key Features

  • Real-time breath detection via device microphone
  • Automatic breath counting and cycle management
  • Audio and haptic feedback for eyes-closed operation
  • Fully customizable parameters (breath count, hold duration, cycles)
  • Completely offline - works in airplane mode
  • Privacy-first - no audio recording or storage
